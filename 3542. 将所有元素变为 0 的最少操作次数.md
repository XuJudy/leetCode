## 3542. 将所有元素变为 0 的最少操作次数

> ##### 题目连接: 
>
> [3542. 将所有元素变为 0 的最少操作次数](https://leetcode.cn/problems/minimum-operations-to-convert-all-elements-to-zero/)

---

##### 解题思路:

##### 单调栈

​	若数字两侧皆为比他小的数，则此数需另外处理，即需多操作一次。

​	令将要插入的数为 a，

​		若 a 大于栈顶元素，直接进行入栈，并记录操作;

​		若 a 小于栈顶元素，则表示此栈顶元素两侧皆为比他小的数，需额外进行一次操作，将栈顶出栈;

​		若 a 等于栈顶元素直接跳过，会跟栈顶一起处理。

​	这样会形成顶大底小的单调栈。

```c++
class Solution {
public:
    int minOperations(vector<int>& nums) {
        
        vector<int> s;  // 栈
        int step = 0;

        for(int a:nums){
            
            while(!s.empty() && s.back()>a)	s.pop_back();
            
            if(a==0)	continue;
            
            if(s.empty() || s.back()<a){
                
                step++;
                s.push_back(a);
            }
        }

        return step;
    }
};
```
