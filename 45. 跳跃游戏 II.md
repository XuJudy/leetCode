## 45. 跳跃游戏 II

> ##### 题目
>
> [45. 跳跃游戏 II](https://leetcode.cn/problems/jump-game-ii/)

##### 解题思想

类似滑动窗口的思想，每次可达区间为`[start, end]`，

在区间中搜寻可到的最远地方`far`，

下次可达区间更改为`[end, far]`。

```cpp
class Solution {
public:
    int jump(vector<int>& nums) {
        
        int step = 0;
        int start = 0, end = 0;
        
        while(end < nums.size()-1){
            
            int far = 0;
            
            for(int i=start; i<=end; i++){
                
                far = max(far, i + nums[i]);
            }
            
            step++;
            start = end + 1;
            end = far;
        }
        
        return step;
    }
};
```
